📱 APP BEHAVIOR ON INSTALLATION
────────────────────────────────────────────────────────────────
When you install and FIRST LAUNCH the app:

✅ Automatically creates these folders on your phone:
   /Android/data/com.example.imageanalysis/files/SFC/
      ├── ReferenceImages/   (empty, ready for your images)
      └── ReferenceData/     (empty, ready for your JSON files)

✅ Automatically saves default paths in Settings:
   • Image Path: [path to ReferenceImages]
   • Data Path: [path to ReferenceData]

✅ Shows toast notification with folder location


📂 WHERE TO FIND FOLDERS ON YOUR PHONE
────────────────────────────────────────────────────────────────
Open File Manager → Internal Storage → Android → data 
→ com.example.imageanalysis → files → SFC

You'll see:
   📁 ReferenceImages/  ← PUT TEST IMAGES HERE
   📁 ReferenceData/    ← PUT JSON FILES HERE

🚨 TROUBLESHOOTING
────────────────────────────────────────────────────────────────
❌ "No reference images found"
   → Make sure images are in ReferenceImages folder
   → Check Settings paths are correct

❌ "No match found"  
   → Similarity is below 75%
   → Try better lighting/angle
   → Ensure reference image is similar

❌ "No data file found"
   → JSON filename must match image name exactly
